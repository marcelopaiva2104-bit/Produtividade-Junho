# Configuração do Supabase

Crie um projeto no Supabase e execute este SQL no editor SQL:

```sql
create table if not exists dashboard_snapshots (
  id bigint generated by default as identity primary key,
  created_at timestamptz not null default now(),
  is_active boolean not null default false,
  payload jsonb not null
);

create index if not exists dashboard_snapshots_active_created_idx
  on dashboard_snapshots (is_active, created_at desc);
```

Crie também um bucket em Storage chamado `produtividade-excel`.

Na Vercel ou Netlify, configure as variáveis:

```text
SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
SUPABASE_BUCKET=produtividade-excel
ADMIN_PASSWORD=
```

Use uma senha forte em `ADMIN_PASSWORD`. A página pública fica em `/` e o painel de upload fica em `/admin`.
